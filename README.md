## 🌟 Projeto em destaque: Neuron

**Neuron** é um jogo de enigmas com foco em **acessibilidade**, criado para incluir jogadores com limitações motoras graves (como tetraplegia) nas dinâmicas do jogo.

- 🎙️ O usuário responde aos desafios por **comando de voz**, processado em **Python** e enviado por porta serial ao **Arduino**.
- 💡 Num **tabuleiro físico**, LEDs representam as "casas" e marcam o avanço dos jogadores a cada enigma resolvido.
- 🧩 Une **hardware, software e acessibilidade** num só projeto.

**Tecnologias do Neuron:** Python · C++ · Arduino Mega 2560 · HTML · CSS · JavaScript

**Status:** em finalização 🛠️
